斐讯R1音箱折腾记录

2022-05-17 Views 技术965字5 min read繁

技术参数

  • 电源输入接口为5.5x2.1mm DC电源插座
  • 底部是2.75英寸低音喇叭，稀土磁铁，最大功率15W
  • 高音软球顶 1.25英寸 喇叭 4欧姆
  • 瑞芯微RK3229处理器和512MB RAM

配网

此为第一步，让音箱连上家里的WiFi

长按 R1 顶部按钮，听到配网模式，R1 会出现一个无密码热点，名称类似 Phicomm_R1_2771

https://github.com/xuconnery/FileOfBossXu/blob/main/Soft/R1%E5%85%8D%E8%B4%A6%E5%8F%B7%E9%85%8D%E7%BD%91.apk

使用方法

连接蓝牙或DLNA

音响开机后按三下顶部的按键，就可以打开蓝牙

为了少操作这3下，直接用“小迅小迅，连接蓝牙”来代替。需要绑定，绑定后，可以使用简单的智障服务，如连接蓝牙，今天天气如何，讲个笑话等。

方案1 修改hosts

在路由器的 hosts 中添加 118.24.249.189 aios-home.hivoice.cn

在老毛子路由器中

在 openwrt 路由器中

打开 CMD 命令行，ping aios-home.hivoice.cn 确定是否劫持成功

当然，不管是网上的118.24.249.189还是 http://47.240.76.176:19999/getUserInfo 都已经失效，可以参考这里自建一个绑定服务器：

https://github.com/xuconnery/FileOfBossXu/blob/main/PDF/%E6%96%90%E8%AE%AFai%E9%9F%B3%E5%93%8D%E8%87%AA%E5%BB%BA%E7%BB%91%E5%AE%9A%E6%9C%8D%E5%8A%A1%E5%99%A8%E6%95%99%E7%A8%8B.pdf

方案2 自建绑定服务器

下载微型WEB服务器+R1配置文件或使用其他web服务器程序都可以，只需将默认的80端口改成19999，把getUserInfo放入网站根目录，确保 http://电脑IP:19999/getUserInfo 能够正常访问（建议关闭电脑防火墙，否则R1可能访问不了）

https://github.com/xuconnery/FileOfBossXu/blob/main/Soft/hfs.exe

https://github.com/xuconnery/FileOfBossXu/blob/main/Soft/R1%E7%BB%91%E5%AE%9A%E6%96%87%E4%BB%B6_getUserInfo

免ROOT关闭R1开机提示音

R1有时突然半夜重启，声音很大，故做此尝试。不知道坛子里是否已经有相关教程，如果有，就以此做个备份吧！

https://www.right.com.cn/forum/thread-4128985-1-1.html

正文开始
1.首先使用adb连接到R1，此步骤不再赘述；
2.使用命令：
adb shell /system/bin/pm hide com.phicomm.speaker.device
注意：直接使用pm是找不到命令的，务必加上绝对路径
3.重启设备，发现开机提示音已经消失！

当然，此方法并非毫无弊端：
此程序禁用后，将不可再使用顶部按钮开启蓝牙，请自行斟酌是否禁用。
若要恢复此程序，则使用命令 adb shell /system/bin/pm unhide com.phicomm.speaker.device
