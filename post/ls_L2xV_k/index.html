魔百盒CM201-2-CH湖北移动版刷机纪要

2022-06-09 Views 技术512字3 min read繁

网上随便一搜有很多魔百盒CM201-2-CH长虹代工的刷机包，比如这个

https://www.znds.com/tv-1171861-1-1.html

但是，做到最后一步，进入recovery刷机界面后，进度条走到三分之一就会闪退然后重启，重启后依旧进入原系统。

网上搜了下，有一些跟我一样的情况，而且貌似都是湖北的，怀疑可能是ROM不对。

偶然间搜到了不刷机，只替换桌面launcher的方法。我想刷机也无非是装几个自己想装的App，能不刷机就不刷机。

具体步骤参考 https://www.znds.com/tv-1144391-1-1.html

如下：

打开隐藏的WiFi功能

点击遥控器设置、进入设备信息栏目

按顺序按： 上上 下下 左右 左右 ok ok （OK按键点击2次，注意是左-右，跟下面的不一样）

这时会弹出提示输入密码：139010086

被隐藏的WIFI功能已经打开，还可以连接5G信号。

永久打开ADB

点击遥控器设置、进入设备信息栏目，停在这个界面等几秒。

然后操作遥控器，按顺序按： 上上 下下 左左 右右 ok ok ok ok ok （ok按键要多按几次）进入了调试模式：

选择adb服务设置，选择永久打开adb服务。关闭盒子在开机就行了

安装、卸载App

可以用当贝的盒子助手 http://www.cnhezi.com/

测试发现当贝桌面不好用，换成据说是更省内存的ATV Launcher，完美运行。

ATV Launcher 下载 https://github.com/xuconnery/FileOfBossXu/blob/main/Soft/ATV-launcher-0.1.9.apk
